## Compile and run API
Build all images:
* Execute: /bin/build.sh 
Push to repository:
* docker tag the-blue-database:latest chariot9/the-blue-database:latest
* docker push chariot9/the-blue-database:latest
Start containers:
* cd compose
* docker-compose up -d
